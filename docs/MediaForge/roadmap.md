# Roadmap

Zurück zur [Masterdatei](MediaForge_Master_Engineering.md).

Die Roadmap ist auf das neue Ziel ausgerichtet: MediaForge baut eine lokale Enhancement Suite für Jellyfin und Audiobookshelf, keine neue Medienplattform.

## V1 - Lokale Basis-Suite

- Docker Compose
- Laravel-App mit PHP 8.4
- PostgreSQL und Redis
- lokales Login
- lokales Dashboard
- Service-Status
- Einstellungen für Jellyfin und Audiobookshelf
- lokale API-Verbindungen
- Grundstruktur für Enhancement-Module
- README und Setup-Dokumentation
- Basis-Backup
- Basis-Health-Check
- erste UI-/UX-Designsystem-Dokumentation

## V2 - Tiefe Integration

- Unified Search
- Unified Metadata Engine
- Storage Analytics
- Health Center
- bessere Import-Workflows
- Audiobookshelf-Kapitel- und CUE-Funktionen
- Jellyfin-Disc-/Codec-Funktionen
- Adult Enhancement über Jellyfin + MediaForge
- Rule Engine
- UI-/UX-Enhancements

## V3 - Professionelle Erweiterungsschicht

- AI Engine
- Plugin SDK
- Developer Center
- Workflow Engine
- Advanced Backup/Restore
- Metadata-Versionierung
- lokale Plugin-Verwaltung
- optionale Upstream-Contributions
- Kompatibilitätsmanagement
- fortgeschrittenes Design-System
- vollständige Premium-Oberfläche
