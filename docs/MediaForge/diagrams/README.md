# Diagramm-Quellen

Dieser Ordner sammelt wiederverwendete bzw. besonders große Mermaid-Quelldateien (`.mmd`) gemäß den Dokumentkonventionen der [Masterdatei](../MediaForge_Master_Engineering.md). Diagramme, die nur in einem Kapitel vorkommen, leben als Mermaid-Block direkt im Kapitel; hier liegt nur, was mehrfach eingebunden oder extern gerendert wird.
